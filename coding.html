2019-2020

The first coding language that I learned is C++. I began learning C++ in 2019 as a hobby outside of school because I wanted to learn how to make games, specifically a game that modelled a Baseball Game Simulator that my dad invented for my brother and I to play. This game had many of the same aspects of a real baseball game and used the stats on TOPPS baseball trading cards to determine the results of any interactions in the game, such as an atbat between a pitcher and batter. With that goal in mind, I set out on my journey to learn how to code in C++.

In the beginning, I tried to learn by watching and following along with The Cherno's YouTube videos, but many of the concepts were too complex for me at the time. As a result, I primarily learned through trial and error and by looking up anything that I didn't know along the way. Obviously, this was not the best or most efficient way to learn, but at the time it worked and allowed me to write my first lines of code. Eventually, I tried to tackle a problem way too advanced for me at the time, 2D graphics in OpenGL, so I got stuck with my C++ coding and took a break from coding for a while.

2021

During my first term at UBC in the fall of 2021, I took my first computer science / coding course, CPSC 110. In that course, I learned a lot of the fundamentals of coding that are transferable to all coding languages such as style, data definitions, functions, and code testing. After completing this course, I had a renewed passion to code again, so I continued my journey with C++, but this time learning from the ground up through a website called Learn C++ as well as through The Cherno's videos mentioned earlier. This time, the learning went much smoother and I was finally able to complete the Baseball Game Simulator that I had begun working on 3 years ago.

2022

After completing the baseball game in early 2022, I started working on other small projects such as Snake, Connect Four and a Typewriter.
